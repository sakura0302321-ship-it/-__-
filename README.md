概要：
画像資料に対して、国際標準規格である IIIF (International Image Interoperability Framework) に準拠した高精度な注釈（アノテーション）を付与・閲覧できるデジタルアーカイブのデモです。
資料の構造化をコンセプトにした直感的なGUI操作により、正確な座標取得から、標準ビューア Mirador での公開までを行うことができます。

主な機能：
4点法による座標取得: 画像上の4隅をタップするだけで、IIIF規格で必須となる xywh（座標とサイズ）を自動算出。
リアルタイム・プレビュー: クリックと同時にCanvas上へ矩形を描画し、保存前にアノテーション範囲を視覚的に確認可能。
演出カスタマイズ（Mirador連携）:サイドバーからの注釈リスト呼び出し。特定領域の強調表示およびテキスト情報の重畳表示。
マルチデバイス・スケーリング: ブラウザ上の表示サイズに関わらず、元画像の解像度に基づいた絶対座標を保持する補正ロジックを搭載。
アーカイブ出力: 生成されたデータはそのままマニフェストファイルに組み込み、世界標準のビューアでの上演（プレイ）に対応。

使用技術：
Language: HTML5 / CSS3 / JavaScript (ES6+)
Libraries:Mirador 3: IIIFシーケンシャルビューアCanvas API: 動的な図形描画および座標制御
Server: Python 3.x (CORS対応独自HTTPサーバー)
Protocol: IIIF Presentation API v3.0

セットアップと実行方法：
サーバーの起動
プロジェクトのルートディレクトリで以下を実行します。
Bash
python server.py

実行
座標取得：http://localhost:8080/viewer.html
アーカイブ閲覧：http://localhost:8080/mirador.html

アセットの準備
任意の画像（sample.jpg）を読み込み、4隅を指定して座標データを取得してください。

こだわりポイント：
UXの追求: 執筆者が資料の構造化に没入できるよう、手動計算を排除したGUIによる自動座標算出機能を構築し、書斎で資料を整理するような利便性を演出しました。
堅牢な状態管理: ページごとの座標データや注釈テキストを辞書型（JSON）で保持し、動的に描画を更新して相互運用性を担保する仕組みを構築しました。
